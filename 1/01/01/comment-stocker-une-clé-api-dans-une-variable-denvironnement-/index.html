Comment stocker une clé API dans une variable d'environnement ?

0001/01/01

Utiliser une API nécessite l’authentification de l’utilisateur à travers une clé API. Stocker la clé API d’un utilisateur dans un fichier R permet de la récupérer à travers d’autres commandes pour modifier des fichiers sur l’API. Cette clé ne sera pas accessible par les autres utilisateurs, évitant ainsi une utilisation abusive de cette clé par d’autre utilisateurs.

Création de l’utilisateur sur l’API

Créer le nom d’utilisateur et le mot de passe de l’utilisateur sur l’application de l’API et générer une clé API.

Stockage de la clé API dans une variable

Créer un fichier texte nommé .Renviron dans le dossier Home de l’ordinateur. Ce fichier est un fichier de configuration de R. Copier cette clé dans une variable : NAME_TOKEN = your_token
